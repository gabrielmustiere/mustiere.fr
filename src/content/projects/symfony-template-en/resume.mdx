A GitHub template that compresses a day of Symfony boilerplate into five commands. Symfony 8 on PHP 8.4, Tailwind 4 without an external bundler, SQLite by
default to avoid any local infra, Mailpit to inspect dev emails, three test tiers wired up (PHPUnit + Playwright), PHPStan at its maximum level from the first
commit, and three MCP servers ready to plug into Claude Code. The defaults are opinionated — no Redis, no API Platform, no design system — so you can clone
and start writing business code within the hour, instead of spending two days uninstalling what you don't want.
