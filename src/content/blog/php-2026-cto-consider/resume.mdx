Fourteen years of PHP/Symfony, around fifty projects, and one observation: in 2026, the ecosystem covers the full spectrum of modern web products. The language
has become strict and performant — typed PHP 8.5, property hooks, pipe operator, JIT IR, stable Fibers. Symfony federates more than fifty decoupled components
(HTTP, Messenger, Workflow, Security, Mailer, Notifier, Uid…) and a complete ecosystem: API Platform for REST/GraphQL APIs, Sylius for e-commerce, EasyAdmin for
the back-office, Mercure for real-time, Live Components for reactive UI, the Hotwire Native bridge for iOS/Android, the Symfony AI Initiative for AI-first apps.
Tooling is industrial: PHPStan level 10, Rector, PHPUnit 13, PHP CS Fixer, `composer audit`, Deptrac. FrankenPHP in worker mode shakes up the infrastructure
equation — 3 to 4× the throughput of PHP-FPM, p95 divided by five, low-cost migration. It's a default option today, not a nostalgic compromise.
